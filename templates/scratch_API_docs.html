Scratch Verify API

Scratch Verify has a public API for usage by Discord bot developers. Using the API, you can query a Discord user ID to check if it's been linked to a Scratch account.

GET http://paric.xyz/scratch/api/user/<userID>

Checks whether a Discord user is verified through their user ID.

Examples

Working example:

> GET http://paric.xyz/scratch/api/user/307778901680062468

{
  "user": {
    "id": 4155652,
    "username": "ComicWorldBros"
  },
  "verified": true
}

Unverified example:

> GET http://paric.xyz/scratch/api/user/451588189015113738

{
  "verified": false
}
